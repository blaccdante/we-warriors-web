## ğŸ”§ Getting Started

### Prerequisites
- Git
- Node.js (for development tools)
- Modern web browser
- Code editor (VS Code recommended)

### Setup
```bash
# Clone the repository
git clone https://github.com/blaccdante/we-warriors-web.git
cd we-warriors-web

# Install development dependencies
```
